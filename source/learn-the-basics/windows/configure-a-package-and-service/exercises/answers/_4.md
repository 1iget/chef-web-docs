```ps
# ~\chef-repo
$ (Invoke-WebRequest localhost).StatusCode
200
```

Here's the code that stops and disables the W3SVC service.

```ruby-Win32
# webserver.rb
service 'w3svc' do
  action [:stop, :disable]
end
```
